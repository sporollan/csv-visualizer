# Oil Data CSV Visualizer

Client tool for processing and visualizing oil-related CSV data.

**Purpose**: Simplify data review for engineering teams by providing immediate visual feedback on oil metrics.

## Quick Use
1. Visit [CSV Visualizer](https://sporollan.github.io/csv-visualizer/)
2. Upload CSV with oil/gas metrics
3. View interactive charts

## Tech
- Vanilla JavaScript + Chart.js
- Client-side processing (data stays in browser)
- Automated deployment via GitHub Pages
